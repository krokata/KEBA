
# KEPA Smart Disability Percentage Finder

Repository που μετατρέπει το ΦΕΚ ΚΕΠΑ σε searchable εργαλείο.

## Περιέχει
- original PDF
- pages.json (όλο το κείμενο)
- percentages.json (γραμμές που περιέχουν ποσοστά αναπηρίας)
- index.html (web search tool)

## Χρήση
1. Άνοιξε το index.html
2. Πληκτρολόγησε πάθηση
3. Θα εμφανιστούν γραμμές με ποσοστά (%) και σελίδα.

Μπορεί να χρησιμοποιηθεί και στο GitHub.
